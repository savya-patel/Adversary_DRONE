 This drone detection system uses YOLOv5 which is a family of object detection architectures. The model is trained on <a href="https://www.kaggle.com/dasmehdixtr/drone-dataset-uav">Drone Dataset</a>. The YOLOv5 detect code is derived from the tusharsarkar3/Detect_Drone repo. 
</p>

## <div align="center">Overview</div>


<details open>
<summary>Install</summary>

Python >= 3.6.0 required with all requirements.txt dependencies installed:
```bash
$ git clone https://github.com/eagleeyeengineering25/Adversary_DRONE.git
$ pip install -r requirements.txt
```
 
</details>

<details open>
<summary>Info</summary>
detect_test_attitude_jetty_flight.py is used for full throttle range.
detect_test_attitude_jetty.py is used for capped throttle (20%) for no prop testing purposes.

</details>
